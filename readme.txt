This App helps to fix the memory issue  and setup the monitor on CPU and memory usage while CopTrax is running together with NetMotion.

1. Run the SetupCPUMemoryMonitor.bat as administrator.
2. If the driver is still there, you are asked to remove it or not.
3. To setup the monitor, type 1. To Stop the monitor type 2.
4. The monitor tries to record the CPU and memory usage of CopTrax and NetMotion every minute. The records are saved as text at C:\CopTrax Support\CPUMemoryMonitor.log.
5. This monitors by default IncaXPCApp.exe (CopTrax), nomtray.exe (NetMotion main), wa_3rd_party_host_32.exe (NetMotion attached, easy to collapse), mesproc.exe (NetMotion related), and mobilecam.exe (Body Camera). You may change the processes to be monitored in config.txt.
